## sg cleanup

Use this to reset what setup did.

### Synopsis

Use this to reset what setup did..

Here's a list of what that is:

- Remove token from git config (use: git town)
- Remove main branch config (use: git town)

```
sg cleanup [flags]
```

### Options

```
  -h, --help   help for cleanup
```
